## 基础要求：

- 数据均为JSON格式的数据
- 需返回请求的数据是否成功以及失败的原因
- 返回的数据放在对象的data属性里面

例如:

```
{
	meta:{
		success:false,// 表示请求是否成功
		reason:0// 表示请求失败的原因,例如：非法文件，这里最好用数字指定某种失败的原因			
	},
	data:{// 返回的实际数据,请求失败则为空对象
		...
	}
}

```


## 各个模块的数据需求


## 课程详情-首页(理论课、实习实训)

- 课程介绍基本信息
